About Taskly

What is Taskly?

Taskly is a simple yet powerful task management application built with Flask and SQLAlchemy. It allows you to create, manage, and organize your daily tasks with ease. Whether you're tracking personal projects or managing work-related tasks, Taskly helps you stay organized and focused on what matters most.

Key Features
  • Create Tasks: Add new todos with a title and detailed description to track what needs to be done.
  • Update Tasks: Edit your existing tasks whenever you need to change details or mark progress.
  • Delete Tasks: Remove completed or unnecessary tasks to keep your list clean and focused.
  • Priority Management: Organize your tasks by priority using the intuitive drag feature. Move tasks up to increase priority or down to decrease it.
  • Sorted Display: All tasks are automatically sorted by priority, so you always see what's most important first.
How to Use
  1. Home Page: Go to the Home page to add new tasks. Enter a title and description, then click Submit.
  2. View Tasks: All your tasks are displayed in a table showing the title, description, and creation time.
  3. Update Task: Click the "Update" button next to any task to edit its details.
  4. Delete Task: Click the "Delete" button to remove a task from your list.
  5. Change Priority: Visit the "Change Priority" page to reorganize your tasks by moving them up or down based on importance.
Technology Stack

Taskly is built using modern, reliable technologies:

  • Flask: A lightweight Python web framework for building the application.
  • SQLAlchemy: An Object-Relational Mapping (ORM) library for database management.
  • SQLite: A simple and efficient database for storing your tasks.
  • Bootstrap 4: A responsive CSS framework for a clean and intuitive user interface.
Get Started

Ready to organize your tasks? Head over to the Home page and start adding your todos today. Your tasks are stored locally and will persist between sessions.
